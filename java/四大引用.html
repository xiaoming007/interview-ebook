四大引用

四大引用

1. 强引用 任何时候都不会被回收，JVM结束，才结束
2. 软引用 SoftReference 内存不足时才会回收，一般用于做缓存
3. 弱引用 WeakReference gc 到来时就会被回收，用来处理内存泄露
4. 虚引用 PhantomReference Gc回收时可得到一个通知
